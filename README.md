# 🧹 Cleaning Service Platform

A modern and responsive **Cleaning Service Website** built using **HTML, CSS, Bootstrap, and JavaScript**. This project represents a multi-page frontend solution for a professional cleaning service business.

🌐 **Live Demo:**  
https://shivanipimple.github.io/Cleaning-service-platform-project/



---

## 📌 Overview

The Cleaning Service Platform is designed to provide users with an easy and interactive way to explore cleaning services, view pricing, check testimonials, and contact the service provider.

The website focuses on **clean UI design, responsiveness, and real-world usability**.

---

## ✨ Features

- 🌐 Fully responsive multi-page website  
- 🎨 Clean and modern user interface  
- 📱 Mobile-friendly design using Bootstrap  
- 🧹 Service listing and details section  
- 🖼️ Gallery to showcase work  
- 💬 Testimonials section  
- 💰 Pricing plans display  
- 📞 Contact page with form  
- 📍 Header with location and contact details  
- 🔗 Social media integration  
- ⚡ Smooth navigation and animations  

---
